# **App Name**: Titan League Simulator

## Core Features:

- League Structure: Configure a 4-division league system with promotion/relegation playoffs and associated logic for season-end tournament qualification.
- Simulation Engine: Simulate match results quickly, driven by team stats and a goal-scoring algorithm weighted by player position.
- Statistical Tracking: Record detailed match stats (goals, assists, clean sheets, cards, MVP) and manage historical data up to 10 seasons.
- Head-to-Head Analysis: Maintain a persistent head-to-head record between teams, automatically updating after each match.
- Player Generation: Automatically generate realistic player names, nationalities, and stats for all teams across the divisions.
- Automated Press Notes Tool: Generate automated textual press notes for key match-ups, powered by an AI tool that uses team logos and key stats to output a 'VERSUS' visual component.
- Team and Player Editing: Provide an interface to edit names, logos, and stats for teams and players.
